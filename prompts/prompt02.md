# Prompt 02 - Quais ferramentas de IA são mais usadas em análise de dados?

## Pergunta

```text
Quais ferramentas de IA são mais usadas em análise de dados?
```

---

## Resposta Obtida no NotebookLM

O NotebookLM apresentou diversas ferramentas de Inteligência Artificial
utilizadas na área de análise de dados, dividindo-as em categorias.

As principais categorias identificadas foram:

- Plataformas de dados unificadas e de nuvem;
- Ferramentas de visualização e Business Intelligence;
- Assistentes de IA e análise conversacional;
- Plataformas no-code;
- Frameworks de Machine Learning.

Entre as ferramentas mencionadas estão:

- Databricks;
- Snowflake;
- Google BigQuery;
- Azure Synapse Analytics;
- Power BI;
- Tableau;
- GitHub Copilot;
- Jupyter AI;
- TensorFlow;
- PyTorch;
- IBM Watson;
- Vertex AI.

A resposta também destacou funcionalidades importantes,
como análise preditiva, consultas em linguagem natural,
geração automática de código, dashboards inteligentes
e automação de processos analíticos.

---

## Análise

A resposta mostrou que o ecossistema de IA para análise
de dados é bastante amplo e vai além das ferramentas
tradicionais de programação.

Foi possível perceber que existem soluções voltadas tanto
para profissionais técnicos quanto para usuários sem
conhecimento avançado em programação, através de plataformas
no-code e interfaces conversacionais.

Outro ponto importante foi a integração entre IA e análise
de dados em ferramentas já muito utilizadas no mercado,
como Power BI, BigQuery e GitHub Copilot.

A utilização de linguagem natural nas plataformas também
mostra como a IA está tornando a análise de dados mais
acessível e produtiva.

---

## Conclusão

As ferramentas de IA estão transformando a área de análise
de dados ao automatizar tarefas, facilitar interpretações
e permitir análises mais rápidas e inteligentes.

Além disso, a integração entre IA, visualização de dados
e machine learning está tornando o trabalho dos analistas
mais eficiente e acessível para diferentes perfis de usuários.
