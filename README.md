# ia-analise-dados-notebooklm
Projeto desenvolvido para o desafio da DIO utilizando o NotebookLM como ferramenta de aprendizagem ativa. O objetivo deste repositório é explorar como a Inteligência Artificial auxilia desenvolvedores e analistas de dados, reunindo fontes confiáveis, testes de prompts, resumos estruturados, glossário técnico e insights obtidos através da IA.
