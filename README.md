# Projeto NotebookLM - IA para Desenvolvedores e Analistas de Dados

## Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio prático da DIO utilizando o NotebookLM como ferramenta de aprendizagem ativa.

O objetivo principal foi estudar como a Inteligência Artificial está transformando o desenvolvimento de software e a análise de dados, explorando ferramentas, automações, impactos no mercado e os desafios relacionados ao uso da IA.

Durante o desenvolvimento do projeto foram utilizadas documentações oficiais, artigos técnicos, pesquisas de mercado e ferramentas modernas de IA para construção de um ambiente de estudo estruturado dentro do NotebookLM.

---

# Objetivos do Projeto

- Compreender como a IA auxilia desenvolvedores de software;
- Estudar aplicações de IA na análise e limpeza de dados;
- Explorar ferramentas modernas de IA utilizadas no mercado;
- Praticar engenharia de prompts utilizando o NotebookLM;
- Organizar informações técnicas em formato de miniguia de estudos;
- Documentar análises, conclusões e aprendizados obtidos durante os testes.

# Principais Aprendizados

Ao longo do projeto foi possível observar que a Inteligência Artificial está cada vez mais presente no cotidiano de desenvolvedores e analistas de dados.

Entre os principais aprendizados obtidos:

- A IA atua em praticamente todas as etapas do desenvolvimento de software;
- Ferramentas de IA aumentam significativamente a produtividade;
- A análise de dados está se tornando mais automatizada e acessível;
- A supervisão humana continua essencial para validação e tomada de decisões;
- A qualidade dos dados impacta diretamente os resultados gerados pela IA;
- Existem riscos importantes relacionados à privacidade, segurança e viés algorítmico.

# Conclusão

O projeto permitiu compreender como a Inteligência Artificial está transformando o desenvolvimento de software e a análise de dados através da automação, aumento de produtividade e geração de insights estratégicos.

Também foi possível perceber que, apesar dos avanços tecnológicos, o fator humano continua fundamental para validação, interpretação e uso ético das soluções baseadas em IA.

O NotebookLM se mostrou uma ferramenta extremamente útil para organização do conhecimento, estudo guiado por IA e consolidação de informações provenientes de múltiplas fontes.
